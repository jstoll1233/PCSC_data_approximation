Method that computes one term of the denominator for both the Barycentric and the lagrange polynomial. Note that the denominator can be divided in one term per interpolation data point (corresponding to one value of k) Note also that the nominator of the polynomial, being different between Lagrange and Barycentric, is defined in the corresponding classes.
